Highlights cliches in your post when you preview it.

== Description ==

This plugin comes with a list of common cliches. When you choose to preview a post it will highlight any cliches that appear with a bright pink colour.


== Installation ==

This plugin requires PHP5

Add to the plugin directory then activate it.

There are no options.
